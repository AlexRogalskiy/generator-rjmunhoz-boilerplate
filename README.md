generator-rjmunhoz-boilerplate
---

Simple yeoman generator to create basic config files for Node projects

## Installation

- `npm i -g yo generator-rjmunhoz-boilerplate`

## Usage

- `mkdir my-wonderful-app`
- `cd my-wonderful-app`
- `yo rjmunhoz`
